# Video Frame Extractor

動画からフレームを抽出してPNG画像として保存できるWebアプリケーション

## 🎬 機能

- ✅ 動画アップロード（ドラッグ&ドロップ対応）
- ✅ カスタマイズ可能なフレーム抽出間隔
- ✅ リアルタイムプレビュー
- ✅ 画像の拡大表示（モーダル）
- ✅ ZIPファイルでダウンロード（カスタマイズ可能なファイル名）
- ✅ 動画の差し替え機能

## 🎨 デザイン

- Appleスタイルのライトデザイン
- Interフォント使用
- スムーズなアニメーション
- レスポンシブデザイン

## 🛡️ セキュリティ

- ファイルサイズ制限（500MB）
- ファイル形式検証（拡張子 + MIMEタイプ）
- 動画長さ制限（30分）
- フレーム数上限（1000枚）
- ファイル名のサニタイズ
- CSP（Content Security Policy）実装
- メモリリーク対策

## 🚀 使い方

### インストール

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## 📦 技術スタック

- React
- Vite
- Tailwind CSS

- Lucide React (アイコン)

## 📝 対応フォーマット

- MP4
- WebM
- OGG
- MOV
- AVI

## ⚙️ 制限事項

- 最大ファイルサイズ: 500MB
- 最大動画長: 30分
- 最大フレーム数: 1000枚
- フレーム間隔: 0.1秒 - 60秒
